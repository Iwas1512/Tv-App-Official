## üöÄ Getting Started

### Prerequisites

- Node.js and npm
- Java JDK 11 or higher
- Maven
- MongoDB

### Installation

1. **Clone the repository**

```bash
git clone https://github.com/Iwas1512/Tv-App-Official.git
cd Tv-App-Official
```

2. **Backend Setup**

```bash
cd backend/movies
./mvnw spring-boot:run
```

3. **Frontend Setup**

```bash
cd frontend
npm install
npm start
```

The application should now be running on [http://localhost:3000](http://localhost:3000).
